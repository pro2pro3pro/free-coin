DiscordCoinSuite - Bot + Web (getcoin, checkcoin, admin)

HOWTO:
1) Fill .env using .env.example
2) npm install
3) npm run register
4) npm run dev  (or npm start)

ENV vars:
- DISCORD_TOKEN
- DISCORD_CLIENT_ID
- DISCORD_CLIENT_SECRET
- DISCORD_WEBHOOK_URL (optional)
- ADMIN_WHITELIST_IDS (csv) and/or ADMIN_SECRET
- SHORTENER_* APIs
- BASE_CLAIM_URL
